# MISUO Sales Data Analysis

## Overview
This project analyzes MISUO sales data using Microsoft Excel to identify sales trends, evaluate business performance, and support data-driven decision-making.

## Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Power Query
- Excel Dashboard

## Project Workflow
- Data Cleaning
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Dashboard Development

## Key Insights
- Cleaned and organized the sales dataset.
- Performed exploratory data analysis to identify sales trends.
- Built an interactive Excel dashboard to visualize key business metrics.
